## v1.2.2-beta.3 — 2026-03-19

### Features
- Multi-user mode with per-user rate limits, role-based ACL, and audit logging
- ImageSender: unified image sending for Feishu, Telegram, Discord, Slack, DingTalk, QQ
- MiniMax model upgraded from M2.5 to M2.7
- `/whoami` command to display user ID for ACL configuration
- `/btw` command to inject messages into busy sessions
- `/dir` command for runtime work directory switching
- Cron job muting with platform wrapper and UI
- Interrupt support (Ctrl+C equivalent for agent sessions)
- CORS support for Bridge API
- Message queuing when agent is busy
- QQ Bot Markdown support

### Bug Fixes
- Workspace session persistence to disk
- Multiple data race fixes (adminFrom, degraded, userRolesMu)
- PendingAcks leak on WeCom disconnect
- Relay timeout returns partial text instead of error
- QQ Bot nil wsConn on reconnect
- i18n: complete translation coverage

### Breaking Changes
- None
